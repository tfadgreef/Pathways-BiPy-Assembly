char* line ( int  label,
char *  content 
)

Fit a string into the Fortran fixed format.

Generates a Fortran fixed format output by distributing the content over multiple lines if lines would exceed the 72 characters allowed by this format. Furthermore the function takes care of the first six control characters. A label will be added if label is not equal to zero.

Parameters
labelLabel to add to the line. 0 for no label.
contentActual code that should be fit into the format.
Returns
The fixed form of the code provided.

Translate a ODE or Jacobian function to Fortran.

Recursively translate the ODE or Jacobian function represented by t to Fortran and print the generated code to the output stream out. jac should be 0 for ODE functions and 1 for Jacobian functions.
